# Uniex Courier — International Logistics Platform

A premium, high-performance web platform for Uniex Courier, built with React, Vite, and Framer Motion. Featuring a cinematic animation system and an automated Smart Assistant for real-time customer support.

## Core Technologies
- **Vite** & **React**
- **TypeScript**
- **Tailwind CSS** & **shadcn-ui**
- **Framer Motion** (Animation System)
- **Lucide React** (Iconography)

## Getting Started

Follow these steps to run the project locally:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd uniex-refresh-glow

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
```

## Features
- **Smart Assistant**: Locally-hosted knowledge base for instant automated responses.
- **Cinematic Animations**: Purposeful motion design across Services, Contact, and About pages.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Brand Excellence**: Integrated high-fidelity carrier logos and professional typography.

## Deployment
Recommended platforms: **Netlify** or **Vercel**.
- **Build Command**: `npm run build`
- **Output Directory**: `dist`
